1. Запуск программы

Программа запускается из командной строки с обязательными параметрами:

program -t <тип> -c <количество> -e <элементы>

-c — количество элементов списка
-e — элементы списка, перечисленные через запятую
-t — тип связного списка:
1 — однонаправленный
2 — двунаправленный
3 — однонаправленный кольцевой
4 — двунаправленный кольцевой

Пример:
program -t 2 -c 3 -e 10,20,30

Если количество элементов не совпадает с -c или переданы неизвестные параметры, программа завершится с ошибкой

2. Меню

После запуска программа выводит меню и ожидает ввода команды
Меню отображается после выполнения каждого действия

1 — добавить элемент  
2 — вывести список  
3 — найти элемент  
4 — удалить элемент  
5 — удалить весь список  
6 — сохранить список в файл  
7 — загрузить список из файла  
0 — выход
